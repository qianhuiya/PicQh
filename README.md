# PicQh
**将GitHub/Gitee/Coding当作自己的图床，上传至代码仓库中,PicQh是基于 GitHub API 的免费、稳定且高效的图床管理神器。**

## 功能特性

- [x] **拖拽图片**进行上传。

- [x] **复制粘贴图片**进行上传。

- [x] **选择文件夹图片**进行上传。

- [x] 图片**重命名**。

- [x] 图片名**哈希化**（保证图片名永不重复）。

- [x] 一键复制 GitHub 和 CDN 图片外链。

- [x] 自动目录（可自动生成存储图片的目录）。

- [x] 图床管理（仓库图片的增删改查）。

- [ ] 图片压缩。

- [ ] 批量上传。

- [ ] 亦接入 Gitee & coding 仓库。

- [ ] 语言国际化。
